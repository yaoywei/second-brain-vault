# 🧠 第二大脑 · 主页

> 这是一个把"我现在想做的事"和"我学到的/读到的/想到的东西"统一管理的地方。
> 目的是：**少回忆，多决策；少重复，多复用**。

---

## 🚀 30 秒上手

1. **每天早上** → 打开 [[10_日记/今日]]，用 [[90_模板/01_日记模板]] 写今天最重要的一件事
2. **任何瞬间冒出来的想法** → 丢到 [[00_收件箱]]，**不在当时整理**
3. **每天晚上** → 花 5 分钟，把 [[00_收件箱]] 清空（用 Claudian 一句话 → 详见 [[90_模板/Claudian话术包]]）
4. **每周日** → 花 30 分钟，跑 [[90_模板/Claudian话术包#4️⃣ 周日整理 SOP|周日整理 SOP]]
5. **每月 1 号** → 做一次月度复盘

## 🔄 CODE 工作流（每天/每周/每月）

### Capture（捕获）— 随时
任何想法、链接、灵感 → 丢到 [[00_收件箱]] 或今天日记。**关键：捕获时不做决策**。

### Organize（组织）— 每天晚上 5 分钟 ⚡
**3 问归类法**（不要想超过 10 秒）

| 第几问 | 问题 | 是 → 放哪 |
|------|------|----------|
| 1 | 这个想法对应正在推进的某个项目吗？ | → [[20_项目]] |
| 2 | 这是长期领域（[[30_领域/]]）的认知补充吗？ | → [[30_领域]] 对应 MOC 末尾 |
| 3 | 只是临时素材/工具链接？ | → [[40_资源/]] 对应分类 |
| ✗ | 三个都不算？ | → 直接删，**别为了"全"硬留** |

**5 分钟完成**：3 问 → 写新笔记（用 [[90_模板/03_资源笔记]] 或 [[90_模板/04_想法模板]]）→ 在原文件头部加归宿标注

🪄 **进阶玩法**：发一句"清空收件箱"给 Claudian，AI 提议去向，你只需 ✅/❌（话术见 [[90_模板/Claudian话术包#1️⃣ 清空收件箱]]）

### Distill（提炼）— 每周日 30 分钟
- 把 [[40_资源/]] 这周新增的笔记，在"原文摘录"下方补上"我的判断"段（**这是 AI 笔记和真正知识库的最大区别**）
- 给每条资源加上 `tags:` 和 `related:` 字段
- 检查 [[20_项目/]] 里停滞超过 1 周的，做"继续/搁置/归档"决策
- 🪄 让 Claudian 跑 [[90_模板/Claudian话术包#2️⃣ 补双向链接|补双向链接]] + [[90_模板/Claudian话术包#3️⃣ 周复盘快照|周复盘快照]]，自动织网

### Express（表达）— 每月一次
- 从 [[30_领域]] + [[50_输出]] 拼出一篇推文
- 让 Claudian 跑 [[90_模板/Claudian话术包#5️⃣ 写公众号稿|写公众号稿话术]] 起中文草稿
- 用 [[99_归档]] 保存已发布的稿件
- 用 [[50_输出/复盘]] 沉淀经验

## 📌 重要原则

> ❌ 不要追求完美分类 — 4 个桶够用，别再细分  
> ✅ 一张笔记一个观点 — 别把 3000 字塞一张卡  
> 🚫 永远不要"先把所有笔记先整理好再说" — 现在就想写就写  
> 🔗 链接 > 分类 — 能 wiki-link 就别开新文件夹  
> 🤖 AI 是队友不是管家 — 让 Claudian 帮你清收件箱、做归纳，**但判断必须你来下**
