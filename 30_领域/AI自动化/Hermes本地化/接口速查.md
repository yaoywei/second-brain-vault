# 🔌 Hermes 本地化接口速查

> 师傅说一句话 → AI 跑哪条 SOP → 一目了然。

## 🗣 师傅说 → AI 跑

| 师傅说（自然语言） | AI 实际跑什么 | 命令 / 动作 |
|------------------|-------------|------------|
| "按 Hermes 出文" | Phase 0-5 完整流程 | `hermes.py preflight && phase1 && style && ...` |
| "Phase 0 跑一下" | 门禁检查 | `hermes.py preflight` |
| "Phase 1 选题" | 联动飞书 13+08 库 | `hermes.py phase1 --topic "..."` |
| "这次用引流文" | 强制引流文 | `hermes.py style --keyword "引流"` |
| "这次写实战复盘" | 强制 dayao 风格 | `hermes.py style --keyword "实战复盘"` |
| "用 n8n 风格" | n8n writer | `hermes.py style --keyword "n8n"` |
| "标题字节校验" | 校验标题 ≤30B | `hermes.py check-title --title "..." --digest "..."` |
| "扫禁用词" | 扫赋能/闭环/... | `hermes.py check-warns --article x.md` |
| "postflight" | 8 步质量门禁 | `hermes.py postflight --output-dir xxx` |
| "推草稿" | Phase 7 | `hermes.py phase7 --html x.html` |
| "看 13 库 schema" | 列字段 | `feishu.py schema 13` |

| "刷新 16 表 schema" | 拉一次缓存 | `feishu.py sync` |

## 🎯 5 大高频场景（最常用）

### 场景 1：写一篇文章

```bash
# 1. 选一条 P0 选题
feishu.py list 08 --filter "优先级,==,P0-先写" --limit 5

# 2. 拿到选题 record_id 后取料
hermes.py phase1 --pool-id recvpdxgZdPwWx

# 3. 写之前判断风格
hermes.py style --keyword "工作流拆解"

# 4. 写完校验
hermes.py check-warns --article draft.md
hermes.py check-title --title "标题" --digest "摘要"
hermes.py postflight --output-dir output/
```

### 场景 2：加一条新原子

```bash
# 1. 看 schema
feishu.py schema 13

# 2. 准备 data，dry-run 试一下
feishu.py add 13 --data '{
  "原子标题": "...",
  "原子类型": "观点",
  "原子内容": "...",
  "来源资料": "...",
  "证据等级": "B-可用观点/方法",
  "Hermes使用状态": "可直接取料",
  "适用栏目": ["工作流拆解"],
  "适用标签": ["B端服务"]
}' --dry-run

# 3. 确认 OK 后去掉 --dry-run
feishu.py add 13 --data '{...}'
```

### 场景 3：选题更新到「写作中」

```bash
feishu.py update 08 recvpdxgZdPwWx --field 生产状态 --value "写作中"
```

### 场景 4：发布后回填 09 队列

```bash
feishu.py add 09 --data '{
  "标题": "...",
  "发布时间": "2026-07-16 09:00:00",
  "阅读": 1234,
  "新增关注": 56,
  "关联选题": ["recvpdxgZdPwWx"]
}'
```

### 场景 5：每周日数据快照

```bash
# 1. 拉一次 schema 缓存
feishu.py sync

# 2. 4 张核心表 stats
for t in 13 08 06 09; do feishu.py stats $t; done
```
