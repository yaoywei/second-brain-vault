## 垂直小号自动化（n8n 工作流）丨实战手册丨2025年12月航海

## 02.用 n8n 工作流搭建选题库

一篇爆款文章的诞生，其要素排序是：选题 > 标题 > 内容逻辑。

选题几乎决定了生死，占据了 70%的重要性。它决定了读者是否（以及为什么）对你的内容感兴趣。

对于刚入门的新手而言，最容易卡住的地方就是选题——“我今天到底该写什么？”

这是因为做垂直内容，我们既要保持“垂直度”，又要保证“高频率”更新，还要尽量蹭热度，这就要求你有一个实时更新的充足且高质量的选题库。

一个理想的选题库，至少需要满足三个条件：

1.

实时性：能抓到当下的热点和趋势。

2.

多样性：避免内容同质化，提供多角度灵感。

3.

高质量：是经过验证的、用户真正感兴趣的内容。

4.

灵活性：我们可以根据自己每个阶段的方向或对标账号，轻松调整对标方向

最常见选题库的搭建方法：

第一种：关注对标账号，“我关注几个头部的对标博主，看他们每天写什么，然后参考数据较好的选题”。

第二种：关注热点趋势，“我定期查看搜一搜垂直领域下的爆款选题，通过当下热点参考选题方向”

这些传统做法当然可行。但是，手动去搜索、整理、归纳……这些重复的体力劳动，不仅会让我们的工作效率极其低下，还会占用本来可以用来投入到其他有价值事情的时间。

本节的核心目的，就是用自动化工作流，帮你解决不知道写什么的问题。

我们将从两个核心角度出发，一步步带你搭建一个自动搭建、自动更新、自动推荐的微信公众号选题库。

我们将通过“n8n + 飞书多维表格 + 邮件通知”构建两个大方向的工作流：

（1） 工作流 1：每日监控对标账号矩阵动态并推送选题分析（日报）

目标：每日自动监控你设定的对标公众号，一旦他们发布新动态，系统将自动抓取并形成选题分析报告。

（2） 工作流 2：每周监控对标关键词领域爆款文章并推送选题分析（周报）

目标：每周自动监控微信“搜一搜”中的特定关键词，帮你筛选出该领域近期的爆款文章选题，为你提供并分析创作灵感。

1.工作流 1：监控对标账号的每日动态（日报）

在上一节“用 n8n 找赛道”中，我们已经筛选并建立了一个“符合筛选标准的对标公众号列表”表格。这个列表是你通过 n8n 自动搜索并且筛选出来的。

添加节点：Code 选择 JavaStript

填写以下代码：

```
// n8n Code 节点，语言选 JavaScript

function cleanMarkdown(markdown) {
  if (typeof markdown !== 'string') return '';

  let result = markdown;

  // 1. 图片：![alt](url)
  result = result.replace(/!\[[^\]]*?\]\([^)]*?\)/g, '');

  // 2. 普通链接：保留文字 [text](url) -> text
  result = result.replace(/\[([^\]]*?)\]\([^)]*?\)/g, '$1');

  // 3. 自动 / 裸链接：<https://xx> 或 https://xx 直接去掉
  result = result.replace(/<https?:\/\/[^>]+>/gi, '');
  result = result.replace(/https?:\/\/\S+/gi, '');

  // 4. 清理多余空格和空行
  result = result
    .replace(/[ \t]+\n/g, '\n')   // 行尾空格
    .replace(/\n{3,}/g, '\n\n');  // 多个空行压成两个

  return result.trim();
}

// 处理所有输入 items
const items = $input.all().map(item => {
  const md = item.json.data || '';     // 你的 markdown 文本字段，如果是 data.content 就改成 item.json.data.content
  const cleaned = cleanMarkdown(md);

  // 你可以覆盖原字段，也可以写到新字段：
  item.json.cleanedData = cleaned;     // 新字段
  // item.json.data = cleaned;         // 若想直接覆盖原来的 data，用这一行替换上面一行

  return item;
});

return items;
```

运行后，可以看到多出来的 CleanedData 是非常整洁易于阅读的文本格式
