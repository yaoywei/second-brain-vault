## 垂直小号自动化（n8n 工作流）丨实战手册丨2025年12月航海

## 03.如何用 n8n 跨平台采集内容

前面我们通过“用 n8n 搭建选题库”学习到如何对标公众号并且定期查询爆款内容获得选题库

但是仅从微信公众号一个平台获取选题内容有极大的限制

（1）选题范围受到局限

（2）重复内容无法建立垂直优势

因此我们要发展多渠道、跨平台收集热门选题的能力，做到让各个平台的热门选题都能为我们所用，并且为 AI 内容二创建立巨大的素材库。

不仅如此，我们还可以通过其他平台的内容直接进行二创，实现源源不断的内容产出流。

1.核心工具解析：MCP 是什么？

在深入实践之前，我们必须理解这项工作流的“秘密武器”——MCP。

许多对 MCP 的解释过于复杂，其核心概念可以更直观地理解。MCP （Model Context Protocol） 是一种协议，它赋予了大型语言模型（如 AI）直接与本地应用和浏览器交互的能力，好比为 AI 装上了可以操作电脑的“手和脚”。

在我们今天的场景中，其工作逻辑是：

1.

我们通过 MCP 工具登录小红书网页版。

2.

在 n8n 中，我们向 AI Agent 发出指令（例如“抓取 10 篇关于重庆火锅的笔记”）。

3.

AI Agent 会调用 MCP，使用 MCP 提供的工具（如搜索、内容获取等接口），通过 API 或浏览器协议来完成搜索与数据提取。

4.

MCP 将执行结果返回给 AI Agent，最终以结构化的数据（JSON）呈现给我们。

核心优势： 这种模式具备强大的稳定性。即使在抓取过程中遇到临时的网络问题或页面结构变化，AI 会根据我们设定的目标，持续调用 MCP 进行尝试与修正，直至成功获取所需数据，极大提升了自动化流程的稳定性。

// 并且把它写回到 parsedObject 的图片字段（images / 图片 / image），不保留数组。

const results = [];

function extractBalancedJSON(str) {
  const s = String(str || '');
  const startIdx = s.search(/[\{\[]/);
  if (startIdx === -1) return null;
  const openChar = s[startIdx];
  const closeChar = openChar === '{' ? '}' : ']';
  let depth = 0;
  for (let i = startIdx; i < s.length; i++) {
    if (s[i] === openChar) depth++;
    else if (s[i] === closeChar) depth--;
    if (depth === 0) return s.slice(startIdx, i + 1);
  }
  return s.slice(startIdx);
}

function buildImagesCommaStringFromObj(obj) {
  if (!obj) return '';
  let imgs = [];
  if (obj.图片) imgs = Array.isArray(obj.图片) ? obj.图片.slice() : [obj.图片];
  else if (obj.images) imgs = Array.isArray(obj.images) ? obj.images.slice() : [obj.images];
  else if (obj.image) imgs = Array.isArray(obj.image) ? obj.image.slice() : [obj.image];

  imgs = imgs.map(u => (u || '').toString().trim()).filter(Boolean);
  imgs = imgs.map(u => u.replace(/[\r\n]+/g, ' '));
  return imgs.join(', ');
}
