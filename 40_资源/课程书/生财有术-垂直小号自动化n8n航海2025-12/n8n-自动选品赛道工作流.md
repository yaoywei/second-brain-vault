## 📝 一句话摘要

用 n8n 把飞书多维表格 + 微信公众号 API + 极致了数据 + DeepSeek 大模型串起来，**自动选品赛道**：API 批量抓取公众号 → AI 自动打标清洗 → 筛选对标账号 → 建立'赛道情报库'

---

## 🔑 关键要点

- **核心数据源组合** = 飞书 Bitable（自建表）+ 极致了数据（公众号数据 API）+ DeepSeek（清洗打标）
- **两套监控并行**：日报（监控对标账号）+ 周报（搜一搜热点）
- 社区节点需要装：Feishu (Lite) + Variable Storage
- **重要：极致了 API 必须自备**——课程里有资源对接，师傅需单独问
