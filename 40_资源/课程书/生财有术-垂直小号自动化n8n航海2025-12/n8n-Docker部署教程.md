# 📘 n8n Docker 部署教程

## 📝 一句话摘要

用 Docker 安装包方式部署 n8n 的完整步骤：下载 Docker Desktop → 拉镜像（提供网盘链接）→ 创建容器 → 端口 5678 映射 → 浏览器访问 http://localhost:5678

---

## 🔑 关键要点

- 镜像包：n8n-ffmpeg.tar.gz（含 ffmpeg，适合公众号视频排版场景）
- 容器名建议：n8n-prod / n8n-test
- 端口：宿主 5678 → 容器 5678
