# 🪄 Obsidian Web Clipper 使用指南

> **目的**：替代 n8n / 自建爬虫，手工把生财有术 / 其他网站的文章**完整剪到 vault**
> **为什么不用爬虫**：师傅已识别反扒强，n8n 抓不到
> **当前状态**：师傅要先看这套手动方案跑不跑得通，再决定要不要自动化

---

## 🛠️ 安装（一分钟）

### 1. 安装浏览器扩展

- Chrome / Edge：搜"Obsidian Web Clipper" → 装官方版
  - https://chromewebstore.google.com/detail/obsidian-web-clipper/cfbcnjggbjdljdgnjjihaipkjcdnklef
- Firefox / Safari：同理去各自扩展商店搜

### 2. 在 Obsidian 里启用"剪藏器"

**这一步是师傅经常忘的！** 没做等于装了白装。

1. 打开 Obsidian → 设置 → 第三方插件 → 已安装插件
2. 找到 **"Web Clipper"**（官方插件，名字里有 O 字母图标）→ 启用
3. （如未自带）从社区插件装上"Web Clipper"或同级
4. **设置 → Web Clipper → 把 vault 链接打开**（关键！）

### 3. 配对（首次）

1. 浏览器点扩展图标 → 它会显示一个配对二维码或短码
2. 在 Obsidian → 设置 → Web Clipper → 粘贴 → 完成
3. 之后每次点扩展就能选 vault + 选模板 → 一键剪藏

## ⚠️ 反扒策略（怎么剪得到）

生财有术反爬机制下，建议师傅这样操作：

### 方案 A：Web Clipper 优先（推荐先试）

- 直接 Web Clipper 看能不能抓到正文
- 失败的话用方案 B

### 方案 B：浏览器手动复制 + Web Clipper

1. 浏览器手动选中文本 → Cmd+C
2. 点 Web Clipper → 选模板 → 模板里有"原文摘录"字段 → Cmd+V 粘贴
3. URL 自动填

### 方案 C：截图 + 文字识别（最后兜底）

如果反爬连"手动选"都失效：
1. Cmd+Shift+4 整页截图
2. 把图片拖到 Web Clipper 的笔记输入框
3. 用 [[image-gen]] 的多模态能力识别后转文字
4. 不优雅但 100% 能拿到内容

## ⚡ 进阶：批量剪藏

如果一次要剪 10 篇：

- 但**重命名规则固定**：`生财-YYYY-MM-DD-编号-标题.md`，方便后续 grep

## 🚧 已知坑

1. **微信公众号**：公众号文章由微信端渲染且禁止外链，Web Clipper 也抓不到全文 → 退到截图方案
2. **需要登录的页面**：Web Clipper 不会替你登录，先登录账号再说
3. **HTTPS 证书错乱**：偶尔发生，重启浏览器即可
4. **CJK 字符乱码**：少数页面 → 报告里手动改成"原文摘录"段
